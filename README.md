# Curso de Personalização de Amigurumis — Área de Membros

Versão corrigida para GitHub Pages.

## Arquivos

Este pacote tem apenas:

- `index.html` — aplicação completa em um único arquivo, com CSS, JavaScript e imagens embutidos.
- `README.md` — este guia.

Não é necessário enviar pasta `assets`.

## Como publicar no GitHub Pages

1. Crie ou abra seu repositório no GitHub.
2. Envie o arquivo `index.html` para a raiz do repositório.
3. Vá em **Settings > Pages**.
4. Em **Build and deployment**, escolha **Deploy from a branch**.
5. Selecione a branch `main` e a pasta `/root`.
6. Salve e aguarde o link ficar disponível.

## Observação

Os vídeos do YouTube, PDFs externos e fontes do Google dependem de internet para carregar.
